# 📊 Global Data Job Market & Salary Analytics Dashboard

An interactive **Data Analytics project** that explores global salary trends and job market insights in the **Data Science, AI, and Analytics industry**.

This project analyzes job salary datasets to uncover patterns related to **job roles, experience levels, company size, remote work trends, and hiring locations** using powerful data analysis and visualization tools.

---

# 🚀 Project Overview

The **Global Data Job Market & Salary Analytics Dashboard** provides valuable insights into the data job ecosystem by analyzing salary datasets from various roles such as:

* Data Scientist
* Machine Learning Engineer
* AI Engineer
* Data Analyst
* Data Engineer
* Software Engineer

The project helps understand **how salaries vary based on experience level, company size, job role, and remote work availability**.

---

# 📊 Key Insights From the Analysis

✔ Identification of **highest-paying roles in Data & AI**

✔ Salary comparison across **experience levels**

✔ Analysis of **remote vs onsite jobs**

✔ Exploration of **top hiring countries**

✔ Salary comparison across **company sizes**

These insights help understand **global demand and compensation trends in the data industry**.

---

# 📈 Dashboard Features

### 💰 Salary Distribution Analysis

Visualizes salary distribution across different data roles.

### 🧑‍💻 Highest Paying Data Jobs

Identifies the **top-paying job roles in the data industry**.

### 📊 Salary vs Experience Level

Shows how salaries grow with experience levels.

### 🌍 Top Hiring Locations

Analyzes which countries are hiring the most data professionals.

### 🏢 Company Size Analysis

Compares salary trends across **small, medium, and large companies**.

### 🏠 Remote Work Trends

Shows the distribution of **remote, hybrid, and onsite jobs**.

# 📊 Dataset Information

The dataset contains information about **global data job salaries and employment characteristics**.

Important columns include:

| Column             | Description                  |
| ------------------ | ---------------------------- |
| work_year          | Year of job record           |
| experience_level   | Experience level of employee |
| job_title          | Job position                 |
| salary             | Salary in local currency     |
| salary_in_usd      | Salary converted to USD      |
| employee_residence | Employee country             |
| company_location   | Company location             |
| remote_ratio       | Remote work percentage       |
| company_size       | Size of company              |

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/codeCraft-Ritik/Global-Data-Job-Market-Salary-Analytics-Dashboard.git
```

Move to the project directory:

```
cd Global-Data-Job-Market-Analytics
```

Install required libraries:

```
pip install pandas matplotlib seaborn plotly streamlit
```

---

# ▶️ Running the Project

Run the Streamlit dashboard:

```
streamlit run dashboard.py
```

The dashboard will open in your browser.
